Typical workflow
1. Create a GitHub repository

On GitHub:

Click New Repository
Name it something like 2d-graph-editor
Create it

GitHub will show commands similar to these.

2. Open terminal in your project folder

Example:

cd path/to/your/project
3. Initialize Git (only once)
git init
4. Add all project files
git add .
5. Make your first commit
git commit -m "Initial project setup"
6. Connect to GitHub repository

Replace with your repository URL:

git remote add origin https://github.com/yourusername/2d-graph-editor.git
7. Push code
git branch -M main
git push -u origin main
Daily workflow

Whenever you make changes:

git add .
git commit -m "Added node creation feature"
git push

Next day:

git add .
git commit -m "Implemented edge drawing"
git push

And so on.
